# Data Import and Insertion

This project provides a Python script for reading data from a CSV file, processing it, and inserting it into a database table using batch transactions. It also includes error handling for reading the CSV and data insertion.

## Prerequisites

Before using this project, make sure you have the following prerequisites installed:

- Python 3.x
- `pandas` library
- `psycopg2` library (for PostgreSQL database)
- A PostgreSQL database connection

You can install the required Python libraries using pip:

```bash
pip install pandas psycopg2 pyspark
